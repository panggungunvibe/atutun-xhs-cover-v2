# 阿囤囤小红书封面 Skill

读取用户已提供的内容 -> 逐步收集人物、辅助素材、标题、封面风格、标题用色偏好 -> 由模型自动决定其余视觉细节 -> 生成可直接用于图片生成模型的封面提示词。

**重要：一次只问一个问题，等用户回答后再问下一个。**

如果用户已经在当前消息提供了某项信息，直接记录并跳过对应问题。用户只需要决定人物、辅助素材、标题、封面风格和标题用色偏好；不要让用户继续做细碎视觉参数选择。

除用户粘贴文章、脚本、选题或上传图片外，只向用户收集下方5步信息。除了第5问只收集固定标题用色偏好外，不要继续追问配色、人物动作、背景、字体、素材用途、构图或装饰偏好；这些答案一律默认是**由模型决定**。

没有被用户明确选择的封面决策项，模型必须根据内容类型自行判断并具体化。最终提示词里不要出现"由模型决定"、"待定"、"可选"或模板占位符。

---

## 五问流程（逐一问，不要一次全问）

**第1问：人物**

要求用户上传人物参考图（图1），不要提供"默认形象"或"通用博主形象"选项。

只问：

请上传图1人物参考图。可以上传1张清晰人脸照，也可以上传2-3张同一人物照片用于保持人物一致性。

如果用户明确说不露脸，也必须要求上传局部出镜参考图，例如手部、半脸边缘或侧脸轮廓。不要凭空生成默认人物。

用户上传图片后，只记录为图1人物参考，不额外追问人物外貌。

---

**第2问：辅助素材**

要求用户上传图2、图3等辅助素材。可以是软件截图、产品图、Logo、工具界面、课程截图、文章截图、数据截图或其他能说明主题的图片。

只问：

请上传辅助素材图。可以上传1张或多张；如果没有辅助素材，回复"无"。

用户上传图片后，只按顺序记录为图2、图3、图4等，不额外追问素材用途。用户回复"无"时，记录为无辅助素材，模型根据主题生成贴纸、图标、清单、箭头和卡片。

---

**第3问：标题**

根据用户已提供的内容、图1人物参考和辅助素材，提炼3个候选封面标题，让用户只选编号：

1. **候选标题A**
2. **候选标题B**
3. **候选标题C**
4. **重新生成3个标题**

不要默认问"是否修改标题"这种开放问题。如果用户主动给出标题，直接使用用户标题。

如果上文和图片信息不足以准确提炼标题，也不要新增开放式追问；仍然基于已有信息给出3个偏通用但可用的标题候选。

标题规则：
- 主标题优先4-10个字
- 可以有副标题，但主标题必须最醒目
- 优先使用口语化强钩子，如"别瞎选了"、"从0搭建"、"全流程公开"、"小白上手"、"到底行不行"
- 测评类必须突出数字，如"5款产品"、"7个维度"
- 教程类必须突出低门槛，如"零基础"、"不用命令行"、"纯小白"

---

**第4问：封面风格**

让用户选择封面风格方向，只给以下选项，让用户只回复编号：

1. **交给模型推荐** — 默认推荐，模型根据标题、人物和辅助素材自动选择最合适的阿囤囤模板
2. **爆款大字压顶风** — 顶部超大标题压住画面，适合强钩子、经验公开、教程入口
3. **巨字拆分冲击风** — 标题拆成多个巨大词块，适合核心概念、强观点、短视频冲击感
4. **小白科普问答风** — 问号、勾选标签和大白话标题，适合给小白讲明白
5. **教程清单风** — 绿色勾选清单明确展示能学到什么，适合部署、安装、上手、全流程
6. **产品测评榜单风** — 大数字、产品卡片、维度标签，适合多产品横评和结论对比
7. **种草推荐风** — 人物点赞、亲切安利、emoji贴纸，适合私藏公开和好物推荐
8. **贴纸拼贴入门风** — 人物、小卡片、截图、Logo围绕拼贴，适合工具组合和工作流搭建
9. **黑底效率工作流风** — 深色工具墙、绿色清单、高对比大字，适合效率方法和工具流公开

不要在这一问里继续追问配色；标题用色只在第5问收集。除标题固定用色偏好外，整体配色由模型根据用户选择的风格、素材复杂度和标题情绪自动决定；标题不必每次都用黄色；没有固定黄色参考时，凡是使用黄色，统一使用柔和浅黄色 #FDFFA7。

---

**第5问：标题用色偏好**

询问用户是否有偏好的、固定的封面标题用色。可以让用户上传过往封面标题颜色截图，用于保持主页配色统一。

只问：

你是否有偏好的、固定的封面标题用色？如果有，可以上传过往封面标题颜色截图，我会按你的固定配色来设定；如果没有，回复"无"，我会根据这张封面的整体效果自动选择标题配色。

用户上传截图后，只记录为标题用色参考，不额外追问色号、品牌色、背景色或其他配色细节。最终提示词必须根据截图提炼出具体标题主色、关键词强调色和描边色；如果能判断近似色号，可以写出近似 HEX 色号。

用户回复"无"时，记录为无固定标题用色，模型根据封面风格、素材复杂度、标题情绪和主页整体感自动选择高对比标题配色。

---

## 自动默认项

除第5问的标题用色偏好外，以下问题不再询问用户，默认全部由模型决定：

- 风格模板：如果用户选择"交给模型推荐"或没有明确选择风格，则在8种风格中自动选择最匹配的一种；如果用户选择具体风格，则按该风格执行
- 标题与整体配色：如果用户在第5问提供固定标题用色或截图，优先沿用其标题主色、关键词强调色和描边逻辑，即使参考色不是 #FDFFA7，也要贴近用户主页原有颜色；如果用户回复"无"，根据封面风格、素材复杂度和主题情绪自动选择高对比亮眼配色；#FDFFA7 是经典默认，不是唯一标题色；没有固定黄色参考时，凡是使用黄色，统一使用 #FDFFA7
- 人物表情和动作：根据标题情绪和内容类型自动选择
- 背景色调：根据素材、主题和封面风格自动选择
- 字体风格：默认综艺感超粗黑体；必要时自动切换为漫画爆款字、圆润粗黑体或科技感粗黑体
- 素材用途：自动判断图2、图3等应作为产品主体、教程证明、背景氛围或贴纸拼贴元素
- 清单、标签、副标题、贴纸、emoji、箭头和装饰：根据选定模板自动生成
- 人物位置、构图比例、遮挡关系和安全区：根据选定模板自动安排

自动决定时必须给出具体结果，例如"人物双手点赞并看向镜头"、"背景为深色工具墙"、"图2作为右侧倾斜产品卡片"，不要把默认项写成抽象描述。

## 出图与复核流程

如果用户要求直接生成图片，默认先生成**1张3:4竖版小红书封面**。除非用户一开始明确要求其他比例，不要同时生成4:3或16:9。

生成3:4封面并展示给用户后，只在用户表达满意、通过、可以、定稿、这版OK等确认后，主动追问一句：

需要我再帮你生成一个4:3和16:9的横版版本吗？

如果用户同意生成4:3和16:9，必须重新做横版构图：4:3和16:9分别适配标题、人物、素材和安全区，不要把3:4竖版直接拉伸、裁切或硬改比例。

在输出最终提示词或调用图片生成工具前，必须做一次内部复核；如果已经生成图片，展示前也必须复核一遍。复核不需要逐条展示给用户，但必须用于决定是否重写提示词或重新生成：

- **比例**：目标比例必须明确且正确；3:4 竖版、4:3 横版、16:9 横版不能混淆，不能接近比例，不能横竖拉伸参考图
- **人物**：真人比例自然，脸和身体不能被拉长或压扁；表情符合标题情绪，不要恐怖、僵硬、苍白、怪异
- **手和肢体**：不要多手、多手指、断手、融合手、错误方向手；如果手势是卖点，必须写清楚手的位置和指向
- **文字**：标题必须清晰可读，不遮挡人物眼睛和嘴巴；重要文字都在安全区内
- **素材**：图1/图2/图3等参考图用途明确，不要把辅助素材误当人物，不要让素材遮挡脸部
- **配色**：如果用户提供固定标题用色，标题主色和关键词色必须贴近用户参考；如果用户无固定用色，标题与背景必须有强对比，根据画面自动选亮眼标题色，避免每张都机械使用 #FDFFA7
- **占位符**：最终提示词里不能保留"由模型决定"、"待定"、"可选"或方括号占位符

如果复核发现比例错误、人物明显变形、手部明显错误、标题不可读或脸部被遮挡，必须先修改提示词并重新生成，不要把失败图当成最终结果交付。

## 通用视觉规范

所有风格都遵守：

- 默认小红书封面构图必须是3:4竖版比例，不接受其他比例或"接近3:4"；只有在用户明确要求或确认扩展横版时，才改为4:3或16:9，并重新适配横版构图
- 真人出镜，人物占画面35%-55%，除非风格另有说明
- 人物边缘有浅黄色（优先 #FDFFA7）、白色或高对比亮色描边，形成明显抠图感
- 主标题是视觉第一优先级，必须超大、清晰、短促
- 主标题必须使用亮眼高对比配色；如果用户提供固定标题用色，优先按用户参考的标题主色、关键词强调色和描边色执行；如果用户回复"无"，默认可选柔和浅黄色 #FDFFA7 或纯白，也可根据画面选择青蓝、亮绿、粉红、橙红、紫色等更适合的标题色；必须有粗黑描边或深色描边
- 标题填充、标题关键词、标题背后的底板/色块/横幅如果使用黄色：用户提供固定标题用色截图时，优先贴近截图中的黄色；没有固定黄色参考时，统一指定为柔和浅黄色 #FDFFA7；不要自行使用荧光黄、高饱和亮黄、金黄或偏橙黄色；如果黄色不适合当前画面，直接换用其他高对比亮色
- 字体使用综艺感超粗黑体、漫画标题字、短视频封面大字
- 常用元素：绿色勾选框、柔和浅黄色 #FDFFA7 箭头、问号、感叹号、emoji、贴纸、小卡片、软件Logo
- 背景可以是室内场景、软件界面、模糊产品图、深色Logo墙、黑灰科技海报或参考素材延展背景，但不能抢标题
- 画面高对比、高密度、强情绪，有"一眼能读懂"的点击感
- 所有重要文字、人物脸部、清单都不超出安全区

避免：

- 不要极简留白
- 不要细字体
- 不要冷淡杂志风
- 不要低对比浅灰字
- 不要全息投影、数据流、粒子特效
- 不要让标题遮挡人物眼睛和嘴巴
- 不要自行用荧光黄、高饱和柠檬黄、金黄或偏橙黄色做标题、关键词或标题底板；如果用户固定标题用色就是这类颜色，必须降低饱和度或通过黑描边、白高光增强可读性
- 不要每张封面都机械使用同一种黄色标题；如果用户没有固定标题用色，标题色必须服务于当前画面

## 生成提示词的核心要求

拿到5步信息后，根据第4问使用用户指定风格，或在用户选择"交给模型推荐"时自动选择最匹配的风格模板，再按第5问处理标题用色，最后按该模板生成提示词。提示词必须做到：

- **人物姿势具体**：说清楚人物在画面哪里、手指向哪里、表情是什么
- **标题层级清楚**：主标题、副标题、清单文字分别写清楚
- **标题配色具体**：明确写出标题主色、关键词色和描边色；如果用户提供标题用色参考，必须贴近参考截图并写出具体颜色，不能强行改成默认黄色；如果用户回复"无"，由模型选择适合整张封面的高对比配色；没有固定黄色参考时，凡是用黄色，都明确写 #FDFFA7；不要把黄色当成唯一可用标题色
- **空间关系清晰**：前景/中景/背景各是什么，哪个遮挡哪个
- **素材指代明确**：直接写"参考图1/图2/图3"，不要复述参考图全部细节
- **风格差异明显**：每种风格有自己的构图、背景、文字位置和装饰逻辑
- **比例写死**：默认最终提示词必须明确写"3:4竖版小红书封面"；如果用户要求4:3或16:9，必须明确写"4:3横版封面"或"16:9横版封面"，并重新安排构图
- **默认项具体化**：所有由模型决定的项都必须在最终提示词中落成具体画面，不保留占位符
- **负面质量约束明确**：写清楚不要多手、不要变形、不要拉伸、不要标题遮脸，尤其是人物参考图为横图时

## 风格一：爆款大字压顶风

顶部超大标题压住画面，人物居中或中下方，适合强钩子标题、经验公开、教程入口。

### 视觉特征
- 标题：顶部占35%-45%，使用与背景强对比的亮眼标题色，常用 #FDFFA7、白色、青蓝、亮绿、粉红，粗黑描边
- 人物：中下方居中，占45%左右，表情强烈
- 背景：室内或主题相关场景，轻微虚化
- 点缀：少量箭头、emoji、贴纸，强化情绪

### 提示词模板

```
参考图1的人物五官特征，保持人物一致性，真人半身出镜。

[具体表情]，人物居中偏下，占画面约45%，[具体手势和身体姿态]，眼神看向镜头，人物边缘有浅黄色 #FDFFA7 描边，形成清晰抠图感。

画面顶部放置超大中文标题"[封面标题]"，字体为综艺感超粗黑体，使用[根据第5问或画面选定的标题主色]填充，粗黑描边，轻微白色高光，文字占据画面上方约40%，有强烈压迫感；标题主色必须和背景强对比，可以是用户固定标题色、#FDFFA7、纯白、青蓝、亮绿、粉红或橙红；如果标题背后需要黄色底板或色块，有固定标题用色参考时贴近参考色，没有固定参考时使用 #FDFFA7，避免自行使用荧光黄或金黄。

如果有副标题，将"[副标题]"放在画面底部，白色粗体，黑色描边，字号小于主标题但仍然醒目。

背景是[背景描述]，轻微虚化，降低细节，让人物和标题成为绝对主体。

画面两侧加入3-5个[主题相关贴纸/emoji/箭头/问号]，只做情绪强化，不抢主标题。

3:4竖版小红书封面，高对比，高密度，强点击感，标题清晰可读，人物脸部不被文字遮挡，所有重要元素不超出安全区。
```

## 风格二：巨字拆分冲击风

把标题拆成2-4个巨大词块，人物被文字包围或夹在中间，适合"产品经理软技能"、"录音工作大法"这类核心概念。

### 视觉特征
- 标题：拆成多个词块，几乎铺满画面
- 人物：夹在字块之间，动作夸张
- 背景：深色街景、室内、模糊工作场景
- 整体：像短视频封面，强冲击、强节奏

### 提示词模板

```
参考图1的人物五官特征，保持人物一致性，真人半身出镜。

[具体表情]，人物位于画面中央，占画面约38%，[具体动作：双手指向上方/双手举起/双手打开]，人物边缘有白色或浅黄色 #FDFFA7 描边。

将中文标题"[封面标题]"拆分成[2/3/4]个巨大词块：[词块1]、[词块2]、[词块3]、[词块4]。

词块使用超粗综艺黑体，使用[标题主色]、纯白和[关键词强调色]交替，全部加粗黑描边。标题主色根据第5问或背景自动选择，可以是用户固定标题色、#FDFFA7、青蓝、亮绿、粉红、橙红或紫色，但必须高对比。词块分别放在画面上方、左侧、右侧和底部，围绕人物排布，部分文字被人物身体轻微遮挡，形成前后层次；如果词块背后有黄色底板或色块，有固定标题用色参考时贴近参考色，没有固定参考时使用 #FDFFA7。

背景是[深色或中性背景描述]，轻微虚化，局部有暗角，让高对比大字更突出。
```

## 通用写提示词规范

1. **先写画面，再写风格**：人物、标题、素材、背景、装饰、构图顺序要清楚。
2. **标题必须原样出现**：不要让模型改写用户确认过的标题。
3. **清单文字必须短**：每条2-6个字，适合封面阅读。
4. **参考图直接编号**：写"参考图1/图2/图3"，不要长篇复述。
5. **少用负面词**：尽量正向描述画面里应该出现什么。
6. **保持阿囤囤风格**：真人出镜、高对比亮色/白色大字、粗黑描边、强情绪、绿色勾选、贴纸感、高点击密度。
